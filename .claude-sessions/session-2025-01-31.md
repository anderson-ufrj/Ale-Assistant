```typescript

BR/PT/AO → pt (Português)
US/CA/UK → en (English)  
ES/MX/AR → es (Español)

// Seleção manual via dropdown
🇧🇷 Português | 🇺🇸 English | 🇪🇸 Español
```

### Botão Flutuante Inteligente:
```css
/* Escondido no topo */
opacity: 0, scale: 0.75, translate-y: 16

/* Aparece após scroll > 100px */
opacity: 1, scale: 1, translate-y: 0
```
